# To-Do List em Python + Flask

![Python](https://img.shields.io/badge/Python-3.11-blue)
![Flask](https://img.shields.io/badge/Flask-2.3-green)
![License](https://img.shields.io/badge/License-MIT-orange)

Uma aplicação web simples para gerenciar tarefas (To-Do List) usando **Python**, **Flask** e **SQLite**.
Permite adicionar, listar e excluir tarefas através de uma interface web.

---

## Funcionalidades

- Listar tarefas existentes
- Adicionar novas tarefas
- Deletar tarefas
- Banco de dados SQLite automático

---

## Pré-requisitos

- Python 3.x
- Pip
- Navegador web
